Welcome to RTDT documentation!
===================================

**RTDT** é apenas um teste para aprender a usar o Read The Docs,
com o intuito de reproduzir essa estrutura de documentação.
O negócio é documentar o que está sendo feito na equipe do
SSL-EL do Pequi Mecãnico da UFG.

Confira a seção :doc:`mecanica` para entender como o robô foi montado,
no que diz respeito a parte física.

Na seção :doc:`eletronica` você vai encontrar informações sobre a parte
eletrônica do robô.

Na seção :doc:`software` você vai encontrar informações sobre o software
que está sendo desenvolvido para o robô.

Na seção :doc:`ai` você vai encontrar informações sobre a inteligência
artificial que está sendo desenvolvida para o robô.

.. note::

   Essa documentação ainda está em desenvolvimento.
